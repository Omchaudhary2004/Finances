Welcome to Finances

This AI-powered stock market prediction app analyzes real-time and historical market data to forecast NIFTY trends (Up/Down) with high accuracy. Using a hybrid model (LSTM + XGBoost), it integrates options chain data, technical indicators, and news sentiment analysis for better predictions. The app collects live market data via the Upstox API and processes it for precise short-term forecasts, including 15-minute candle trends. Designed for traders and investors, this tool helps in making informed decisions based on data-driven insights.
